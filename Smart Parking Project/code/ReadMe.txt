Parking Automatique – Arduino
À propos
Ce projet est un système automatique de parking utilisant Arduino. Il permet de :

Détecter une voiture à l’entrée ou à la sortie,

Ouvrir et fermer la barrière automatiquement,

Compter le nombre de places disponibles,

Afficher ce nombre sur un afficheur 7 segments.

Contexte
J’ai créé ce projet en 2019 dans le cadre d’une formation en robotique avec l’Association Jeunes Sciences Zarzis (AJSZ).
Ensuite, j’ai présenté ce projet à la foire régionale de Gabès, où il a été bien accueilli.

Pourquoi ce code ?
Je n’ai plus le code original, donc j’ai recréé le code à partir de mes souvenirs et de quelques notes. Le but est de garder la même idée simple pour qu’il soit facile à comprendre.

Matériel utilisé
Arduino Uno,

2 capteurs LDR (pour détecter les voitures),

2 servomoteurs (pour les barrières),

1 afficheur 7 segments,

Résistances et fils de connexion.



Auteurs
